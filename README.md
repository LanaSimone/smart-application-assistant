# Job Tracker

A full-stack job application tracker built with React, Node.js, Express, and SQLite.

## Features
- Add new job applications
- View saved applications
- Update application details
- Delete applications
- Track application status through a simple interface

## Tech Stack
- React
- Node.js
- Express
- SQLite

## Project Status
This project is currently in progress. Current functionality includes full CRUD operations for job applications, and additional improvements are being added over time.
